== Installation ==

1. Upload the `relap` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your <a href="http://relap.io/widgets">http://relap.io</a> token 
4. Go to the widgets page, and place widget on sidebar.
